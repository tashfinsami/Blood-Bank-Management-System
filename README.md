# Blood Bank Management System

A desktop application built using **Java Swing** for the graphical user interface and **MySQL** for the database. This system is designed to help blood banks, hospitals, and administrators efficiently manage blood donors, donations, and inventory.

## Features
- Add, update, and delete donor information
- Track blood donations and current blood stock
- Search and filter donors by blood group or location
- Generate reports on blood inventory and donor history
- User-friendly interface with Java Swing GUI

## Technology Stack
- **Frontend:** Java Swing
- **Backend:** Java
- **Database:** MySQL (with JDBC for connectivity)

## Setup & Usage
1. Create the MySQL database using the provided SQL scripts.
2. Configure the database connection in the Java application.
3. Run the `Main.java` file to launch the application.

## Purpose
This project demonstrates a practical **desktop-based management system**, focusing on CRUD operations, database integration, and GUI development in Java.
